## Python OOP – Hinglish Short Notes for Revision

### 1. **Class & Object**
- **Class**: Blueprint/template. Jaise car ka design.
- **Object**: Class ka instance. Jaise actual car.
```python
class Car:
    pass
my_car = Car()  # Object banaya
```

### 2. **Constructor (__init__)**
- Object create hone pe automatically call hota hai.
- Instance variables initialize karta hai.
```python
def __init__(self, name):
    self.name = name
```

### 3. **Instance vs Class Variables**
- **Instance**: Har object ka apna (self.name).
- **Class**: Saare objects ka same (Car.wheels = 4).

### 4. **Methods Types**
- **Instance method**: `self` parameter leta hai.
- **Class method**: `@classmethod`, `cls` parameter.
- **Static method**: `@staticmethod`, self/cls nahi chahiye.

### 5. **Encapsulation**
- Data hiding. Private variables: `_variable` (protected), `__variable` (private).
```python
self.__balance = 1000  # Private
```

### 6. **Inheritance**
- Parent class se child class properties inherit karta hai.
```python
class Animal:  # Parent
class Dog(Animal):  # Child
```

### 7. **Polymorphism**
- Same method name, different behavior. Method overriding.
```python
def speak(self):  # Har class me alag implementation
```

### 8. **Abstraction**
- Essential features dikhana, implementation hide karna.
- Abstract classes: `from abc import ABC, abstractmethod`

### 9. **Method Overriding**
- Child class me parent ka method redefine karna.

### 10. **super() Function**
- Parent class ke methods access karne ke liye.

### 8. **Class method vs Static method vs Instance method**
```python
class MyClass:
    class_var = "Class Variable"
    
    def instance_method(self):  # Instance method
        return self.class_var
    
    @classmethod
    def class_method(cls):  # Class method
        return cls.class_var
    
    @staticmethod
    def static_method():  # Static method
        return "Static method called"
```
**Concepts:** Method types, Decorators

### 9. **Property decorator use karo**
```python
class Circle:
    def __init__(self, radius):
        self._radius = radius
    
    @property
    def area(self):
        return 3.14 * self._radius * self._radius
    
    @area.setter
    def area(self, value):
        self._radius = (value / 3.14) ** 0.5

c = Circle(5)
print(c.area)  # Getter
c.area = 100   # Setter
```
**Concepts:** Properties, Getter/Setter

### 10. **Operator overloading kaise karte hain?**
```python
class Point:

        self.x = x
        self.y = y
    
    def __add__(self, other):
        return Point(self.x + other.x, self.y + other.y)
    
    def __str__(self):
        return f"({self.x}, {self.y})"

p1 = Point(1, 2)
p2 = Point(3, 4)
p3 = p1 + p2  # __add__ method call hoga
print(p3)  # (4, 6)
```
**Concepts:** Operator overloading, Magic methods

### 12. **Diamond problem kya hai Python me?**
```python
class A:
    def method(self):
        print("A")

class B(A):
    def method(self):
        print("B")

class C(A):
    def method(self):
        print("C")

class D(B, C):
    pass
```

### 13. **Context manager banao (__enter__, __exit__)**
```python
class FileManager:
    def __init__(self, filename):
        self.filename = filename
    
    def __enter__(self):
        self.file = open(self.filename, 'w')
        return self.file
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        self.file.close()

# Usage
with FileManager('test.txt') as f:
    f.write('Hello')
```
**Concepts:** Context managers, Magic methods

### 14. **Singleton pattern implement karo**
```python
class Singleton:
    _instance = None
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance

s1 = Singleton()
s2 = Singleton()
print(s1 is s2)  # True
```
**Concepts:** Design patterns, __new__ method

### 15. **Metaclass ka basic example**
```python
class MyMeta(type):
    def __new__(cls, name, bases, attrs):
        # Har method name ko uppercase kar do
        new_attrs = {}
        for key, value in attrs.items():
            if callable(value):
                new_attrs[key.upper()] = value
            else:
                new_attrs[key] = value
        return super().__new__(cls, name, bases, new_attrs)

class MyClass(metaclass=MyMeta):
    def hello(self):
        print("Hello")

obj = MyClass()
obj.HELLO()  # Method name uppercase ho gaya
```
**Concepts:** Metaclasses, Advanced OOP
